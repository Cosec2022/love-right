# Love Right story workflow

For every new or substantially revised story, start with [LR_STORY_CREATION_PROTOCOL.md](./LR_STORY_CREATION_PROTOCOL.md). It is the executable, authoritative workflow for Codex and human contributors.

The supporting skills are reference roles within that protocol: Brief Freezer, Scene-Driven Choice Designer, Writer Brain, Independent Reviewer, Revision Controller, Outcome/Hidden Ending Designer, Measurement Annotator, and Release Gate. Measurement and product integration occur only after Narrative Approved; neither may direct plot toward archetype quotas.
