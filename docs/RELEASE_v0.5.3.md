- 结果标题不再只显示“会把细节记很久的人”，而是直接显示为两层：
  - 你是一个
  - 会把细节记很久的人
- 前缀直接指向答题者，核心短标题仍保持大字号和记忆点。

## Story 06 重构

《凌晨一点，她只给你三分钟》改为一条稳定主线：

1. 先用匿名语音建立习惯与声音吸引；
2. 再让唱片店店主进入现实线；
3. 身份线索逐步重合，不突然确认；
4. 隐私选择主要改变台词与信任，不制造长期世界线；
5. 仅在第15幕打开四条身份处理路线；
6. 每条路线在第16幕被完整承接后，再回到共同的现实关系问题。
