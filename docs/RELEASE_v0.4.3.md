# Love Right v0.4.3

## Romantic Ending & Emotional Indices

本版根据首轮用户反馈调整结果页：保留故事性和详细解读，同时恢复少量、可区分的指数。

### 结果页顺序

1. 故事结局
2. 核心结果名称与矛盾
3. 三个关键关系结论
4. 六项情感维度指数
5. 四个关系动作
6. 三个关键剧情证据
7. 深层解读、过往投射、未来三幕与适配对象
8. 折叠的16维研究数据

### 名称调整

- `最想被坚定选择的人` → `想被认真放在心上的人`

### 六项情感指数

- 心动浓度
- 靠近勇气
- 确定需求
- 清醒判断
- 自我边界
- 长期投入

六项指数来自现有8个投射 meter，经加权后在单个用户本篇故事内做自适应对比拉伸。它保留底层高低顺序，但不代表和其他用户比较的绝对分数。前端明确显示这一说明。

### 结局重写

四篇故事共16个 outcome 全部重写为更有画面感的结局，并将男/女主魅力偏好作为第二段情绪回响。不同选择仍然保留不同方向，不强行把拒绝、暂停或离开写成失败。

### 验证要求

- 4篇故事均保持18次选择完成
- 每份结果必须有6个情感指数
- 指数值范围18—92
- 单份结果最大值与最小值至少相差20
- 结局必须位于心理解读之前
- 每个结局主段落至少80个字符
- 原有结果分布审计继续通过
