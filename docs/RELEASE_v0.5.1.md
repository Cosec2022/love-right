# Love Right v0.5.1 · Emotional Continuity

本版集中修复用户反馈中的“上一幕有情绪，下一幕像没发生过”。

- 新增六项关系温度状态：靠近、信任、戒备、受伤、张力、修复。
- 选择不再只进入人格空间，也会改变当前关系气候。
- 88 个高情绪选项增加专属余波文本；下一幕先承接刚刚的态度，再推进剧情。
- 冷淡、拒绝、质疑、退出不再被解释成欲擒故纵。
- 冲突后的剧情必须保留问题，直到出现解释、接受或明确退出。
- 返回上一幕会同步恢复人格向量、关系温度和情绪余波。
- 同时收录两篇男性向故事：竞争合作与匿名双身份。

底层 16D 人格空间继续负责结果准确度；关系温度只负责剧情连续性，两者分开计算。
